# Contribuindo com o Calcmed

Obrigado pelo interesse em contribuir! Siga as orientações abaixo.

## Como contribuir

1. Faça um fork do repositório e crie um branch a partir de `main`:
   ```bash
   git checkout -b minha-contribuicao
   ```

2. Faça suas alterações em `calc.py` (ou nos demais arquivos relevantes).

3. Teste manualmente executando `python calc.py` e verificando os cálculos.

4. Abra um Pull Request descrevendo o que foi alterado e por quê.

## Boas práticas

- Mantenha a interface em português (pt-BR).
- Não quebre a compatibilidade com Python 3.8+.
- Qualquer nova fórmula ou cálculo deve ser verificado com fontes farmacológicas confiáveis.
- Evite adicionar dependências externas sem necessidade clara.

## Reportando problemas

Abra uma [issue](../../issues) descrevendo o problema, os valores de entrada utilizados e o resultado esperado vs. obtido.
